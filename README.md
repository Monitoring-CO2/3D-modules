## Software used

Free and open source software : FreeCAD

## Tools and screws

- 1x Screwdriver

## Assembly

- Red : M3 12mm screw
- Purple : M3 16mm screw
- Green : M2 8mm screw
- Yellow : M3 Nut
- Blue : M3 20mm Threaded spacer female-female
- Pink : M3 30mm Threaded spacer male-female
- ( PCB or 2 washer between the two treaded spacer)
